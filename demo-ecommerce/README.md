# Demo E-Commerce Website with HoneyCloud Integration

A professional demonstration of security monitoring integration for final-year projects and technical interviews.

## 🎯 Purpose

This project demonstrates:
- How production applications integrate with SIEM/security monitoring systems
- Honeypot endpoint design and implementation
- Real-time security event reporting
- Security-aware system architecture

## 📋 Prerequisites

- Python 3.8+
- HoneyCloud platform running on `http://localhost:8000`
